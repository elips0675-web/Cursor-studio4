# Реализованный функционал в приложении SwiftMatch

Этот документ содержит обзор ключевых функций, которые были реализованы и настроены в приложении.

---

### 1. Основной функционал

*   **Профили пользователей:** Создание, редактирование и просмотр профилей с фотографиями, личной информацией (возраст, рост, город), интересами, био и званиями.
*   **Аутентификация:** Реализован полноценный вход через Google с помощью Firebase Authentication, а также эмуляция входа по Email и телефону. Профиль пользователя хранится в базе данных Firestore.
*   **Лента Поиска:** Единый центр для просмотра анкет. Поддерживает два режима:
    *   **Люди рядом:** Отображает ближайшие анкеты.
    *   **Автопоиск:** Умный алгоритм, который автоматически подбирает релевантные анкеты на основе данных вашего профиля и показывает их в режиме свайпов.
*   **Система мэтчей:**
    *   При взаимном лайке появляется анимированное окно "Это совпадение!".
    *   **AI-анализ совместимости:** В окне мэтча отображается краткий анализ совместимости, сгенерированный искусственным интеллектом.
*   **Чаты:**
    *   Полноценный функционал чатов с matched-пользователями.
    *   **AI-помощник:** Генерация "icebreakers" (фраз для начала разговора) на основе интересов и био собеседника.
*   **Видеозвонки:** Реализована функция видеозвонков с доступом к камере устройства, что делает общение еще более живым.
*   **Активность:** Раздел для просмотра лайков, визитов профиля и новых мэтчей.

### 2. Безопасность и Модерация

*   **Антимат (Anti-profanity):** Внедрена система автоматической фильтрации сообщений в чатах. Сообщения, содержащие нецензурную лексику, спам, политические лозунги или подозрительные ссылки, блокируются до отправки.
*   **Система жалоб:** Реализован механизм «Report» для пользователей. Можно пожаловаться на анкету или чат, выбрав причину (спам, оскорбления, фейк). Жалоба мгновенно попадает в панель администратора.
*   **Защита данных:** Использование Firebase Security Rules обеспечивает доступ к данным только авторизованным владельцам.

### 3. Уникальные AI-функции (Genkit)

*   **Генерация био для профиля:** AI помогает пользователям создать креативное и привлекательное описание профиля.
*   **Анализ совместимости:** При мэтче AI анализирует профили двух пользователей и предоставляет краткую выжимку об их совместимости.
*   **Icebreakers для чатов:** AI предлагает персонализированные фразы для начала диалога на основе настроения.

### 4. Приватность и Монетизация

*   **Режим "Инкогнито":** Позволяет пользователям просматривать чужие профили, не оставляя следов в разделе "Визиты".
*   **Монетизация "Автопоиска" и "Буста":** Доступ к премиум-функциям предоставляется либо за просмотр рекламы, либо за небольшую плату.

### 5. Администрирование и Модерация (Раздел `/admin`)

*   **Дашборд:** Отображение ключевых метрик: общее количество пользователей, онлайн, новые регистрации и мэтчи.
*   **Управление пользователями:** Таблица со списком всех пользователей с возможностью редактирования данных и удаления.
*   **Система обработки жалоб:** Администратор видит список всех тикетов от пользователей, может просмотреть профиль нарушителя и принять решение о блокировке.
*   **Управление функциями (Feature Flags):** Включение и отключение ключевых функций приложения (видеозвонки, AI-фичи, страница групп) в реальном времени.
*   **Монетизация:** Настройка ID рекламных блоков для Google AdMob и Яндекс.Директ.
*   **Рассылки:** Интерфейс для создания и отправки сообщений всем пользователям приложения или по Email.
*   **Контент-менеджмент:** Редактирование глобальных списков интересов, целей и уровней образования.
