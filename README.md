# SwiftMatch - AI-Powered Dating Platform (White-Label MVP)

Это полноценное веб-приложение (MVP), построенное на базе Next.js 15, React 19 и Firebase, упакованное как готовое бизнес-решение.

## 🚀 Ключевые особенности
- **AI-Интеграция (Genkit)**: Анализ совместимости пар, генератор био и персонализированные "icebreakers".
- **White-Label Ready**: Архитектура позволяет запустить клон приложения в новом регионе за 60 минут (инструкция в `docs/RUNBOOK.md`).
- **Бизнес-аналитика**: Встроенный дашборд для инвесторов с графиками роста (DAU, Matches, Retention).
- **Безопасность**: Автоматический фильтр нецензурной лексики (Anti-profanity) и система модерации жалоб.
- **Юридическая база**: Готовые шаблоны Privacy Policy, Terms of Service и GDPR/152-ФЗ согласия.

## 🛠 Технологический стек
- **Frontend**: Next.js 15 (App Router), TypeScript, Tailwind CSS, ShadCN UI, Framer Motion.
- **Backend**: Firebase Authentication (Google/Email), Firestore (Real-time DB).
- **AI**: Google Gemini 2.5 Flash через Genkit.
- **QA**: Структура для Jest (Unit), Playwright (E2E) и K6 (Load Testing).

## 📈 Монетизация
Проект поддерживает три модели дохода:
1. **Premium подписки** (безлимит лайков, инкогнито).
2. **Разовые покупки** (Бусты, Автопоиск).
3. **Rewarded Ads** (Google AdMob, Yandex Ads).
